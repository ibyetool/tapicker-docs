# 基础操作

编辑器主要有以下几个基础操作：

1. **切换大小** - 点击这里可以展开或收缩编辑器
2. **切换位置** - 点击这里可以切换编辑器的位置
3. **配方名称** - 点击这里可以设置配方名称，可见性等
4. **帮助文档** - 点击这里打开帮助文档
5. **关闭窗口** - 点击这里将关闭编辑器（如果配方未保存，将丢失）
6. **取消** - 等同于“关闭窗口”
7. **保存** - 创建或更新配方
