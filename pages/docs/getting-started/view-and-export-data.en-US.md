# View And Download Data

## 1.View Data

Open the `Data` panel, find `@official/amazon-knapsack#T492`, and click the `View` button to view the data.

## 2.Export Data

Click the `Export` button to export data, and support common file formats, such as: .json, .csv, .xls, .xml.
